    2. Main Features

    Activity Timeline/Log: Displays activity logs such as employee attendance, sales from the cashier app, employee tasks, and more—adjusted based on the user's role.

    Attendance with Facial Recognition & Geolocation: Employees can check in using face detection and location verification within a configurable radius.

    Off-Site Attendance Requests: If work requires attendance outside the workplace (e.g., fieldwork), employees can request attendance via the "Off-Site Attendance" button with a clear explanation. If the explanation is unclear, the admin may reject the request and mark the attendance as invalid.

    Face Verification on Account Registration: Facial detection must be verified during account registration.

    Individual Activity Page per Employee

    Payslip Page

    Leave Request Page

    Loan Request Page

    Overtime Request Page

    Employee Attendance Report Page

    Daily Task Page

    Anti-Duplicate Attendance System: Employees can only check in once per session. For example, if an employee checks in at 08:10 AM, they cannot check in again that day—only check out when leaving work. The system blocks multiple check-ins on the same day.

    3. Miscellaneous

    Account Switching (Up to 3 Accounts): Allows up to 3 accounts for login on a single device. Useful for employees having device issues, enabling them to log in on a coworker’s device. Since attendance uses facial detection, it cannot be manipulated.
